# 聚宽量化交易策略 (JoinQuant Stock Strategies)

基于聚宽 (JoinQuant) 平台的 A 股量化交易策略

## 📈 策略列表

### 1. 分钟级别动量策略 (`joinquant_momentum_strategy.py`)

**策略特点：**
- 时间级别：分钟线（30 分钟调仓一次）
- 选股逻辑：选择过去 60 分钟涨幅前 10 的股票
- T+1 处理：记录买入时间，次日才可卖出
- 风险控制：止损 5% / 止盈 10%
- 仓位管理：单只股票最大仓位 25%

**适用场景：** 短线交易、日内交易

**回测建议：**
- 时间：近 3 个月
- 初始资金：10 万 +
- 频率：分钟线回测

---

### 2. 日线级别动量策略 (`joinquant_daily_strategy.py`)

**策略特点：**
- 时间级别：日线（每日 14:50 调仓）
- 选股逻辑：选择过去 20 日涨幅前 10 的股票
- T+1 处理：记录买入日期，次日才可卖出
- 风险控制：止损 8% / 止盈 20%
- 仓位管理：单只股票最大仓位 10%

**适用场景：** 中线交易、趋势跟踪

**回测建议：**
- 时间：1 年以上
- 初始资金：10 万 +
- 频率：日线回测

## 🚀 使用方法

### 1. 在聚宽平台使用

1. 登录 [聚宽官网](https://www.joinquant.com/)
2. 进入 **策略研究** → **创建策略**
3. 复制策略代码到编辑器
4. 设置回测参数
5. 运行回测

### 2. 回测参数设置

**分钟线策略：**
```
回测时间：2025-12-15 至 2026-03-15（近 3 个月）
初始资金：100000
频率：分钟线
基准：沪深 300 (000300.XSHG)
```

**日线策略：**
```
回测时间：2025-01-01 至 2026-03-15（1 年 +）
初始资金：100000
频率：日线
基准：沪深 300 (000300.XSHG)
```

## ⚙️ 策略参数说明

### 分钟线策略参数
```python
g.STOCK_POOL_SIZE = 10          # 持有股票数量
g.HOLD_PERIOD = 60              # 动量周期（60 分钟）
g.REBALANCE_MINUTES = 30        # 调仓间隔（30 分钟）
g.STOP_LOSS = 0.05              # 止损 5%
g.TAKE_PROFIT = 0.10            # 止盈 10%
g.MAX_POSITION = 0.25           # 单只仓位 25%
```

### 日线策略参数
```python
g.STOCK_COUNT = 10              # 持有股票数量
g.MOMENTUM_DAYS = 20            # 动量周期（20 日）
g.STOP_LOSS = 0.08              # 止损 8%
g.TAKE_PROFIT = 0.20            # 止盈 20%
g.MAX_POSITION = 0.10           # 单只仓位 10%
```
